1. Put your songs in the PLAYLIST folder
   - Supported formats: .mp3, .m4a, .flac, .opus, .ogg, .alac, .wav, .wma

2. Double-click UPDATE_PLAYLIST.bat
   - This will convert all songs to a roblox-friendly format (44100hz libvorbis ogg) and store them in the playlist_cache folder

3. Double-click RUN.bat
   - You may be asked to setup 'winfsp'. This is the tool used to create a virtual disk drive that allows for overriding the grace soundtrack with custom songs.

4. Join/Rejoin the Grace Lobby. You should see 'Piped cache file' in green text when the music is successfully patched

5. In-game, supported tracks will be replaced. (Ultrasonic Faith, Clocktower Mayhem, etc)

6. Before closing the Grace Runtime Audio Patcher, press CTRL+C in the patcher console. This is necessary in order to revert edits made to the Roblox sound cache. (if you happen to forget, just start and close the patcher, which will resolve any issues)




BONUS: Naming a file (any file extension) in PLAYLIST the following will override:
  special_lobby -> Override falling down elavator music
  special_carnation -> Override carnation chase sfx
  special_dyson_vacuum -> Override elkman/soulsucker sfx (broken!!!)
  special_random_ass_scream -> Override a random scream sound effect
  special_selectall -> Override all sound effects, takes all priority, works in any and all roblox games
